

# Objetivo do Projeto 
Criar o protótipo de um site para **"Confeitaria Sempre Doce"** onde seja possível mostrar seus produtos, as informações da empresa e um formulário de contato. 

## Requisitos:
O site deve ter quatro páginas, início, sobre, cardápio e contato. As páginas devem ser intuitivas com design que facilite a usabilidade onde o cliente facilmente consiga encontrar as informações que deseja com poucos cliques. 

## Ferramentas:
Draw.io, Figma ou qualquer outra ferramenta de prototipagem de sua preferência.
